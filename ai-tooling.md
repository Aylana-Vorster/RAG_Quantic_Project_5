# AI Tooling Documentation

## Tools Used
- **LLM Engine**: Groq (Llama-3 series).
- **Embeddings**: HuggingFace Sentence-Transformers.
- **Orchestration**: LangChain.
- **Vector Store**: ChromaDB.

## Justification
- **Groq**: Chosen for low-latency inference on open-weight models.
- **LangChain**: Standardized interface for building RAG pipelines.
- **ChromaDB**: Lightweight, local-first vector storage.
- **Sentence-Transformers**: Open-source local embeddings ensure privacy and performance for local ingestion.
